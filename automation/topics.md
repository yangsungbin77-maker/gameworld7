# 발행 대기 주제 큐

자동 발행 시스템이 위에서부터 `- [ ]` 항목을 하나씩 꺼내 글로 만듭니다.
발행되면 `- [x]`로 바뀝니다. 새 주제는 아래에 `- [ ]` 형식으로 계속 추가하세요.
한 줄에 주제 하나. 필요하면 ` — ` 뒤에 방향(앵글)을 덧붙여도 됩니다.

- [ ] 입문자에게 추천하는 협동 멀티플레이 게임 — 친구와 같이 시작하기 좋은 것들
- [ ] 게임 멀미(시뮬레이터 멀미) 줄이는 설정과 습관
- [ ] 스팀 덱·휴대용 게이밍 PC, 사기 전에 알아야 할 것
- [ ] 오픈월드 게임에서 길 잃지 않고 끝까지 즐기는 법
- [ ] 고사양 그래픽카드 없이도 즐길 수 있는 가벼운 명작 게임
- [ ] 장르별로 추천하는 입력기 — 게임패드 vs 키보드·마우스
- [ ] 인디 게임 고르는 기준 — 숨은 명작 찾는 법
- [ ] 멀티플레이 매너와 보이스챗 에티켓
- [ ] 세이브 데이터·계정 안전하게 관리하는 법
- [ ] 게임 하나를 오래 즐기는 사람들의 습관
