# 03. 사이트 소개 (CTA·정체성)

## 한 줄 소개
**게임월드7(gameworld7.com)** — 게임 뉴스·공략·리뷰를 가리지 않고, 추천·방법·정책·사양을 "직접 파보는 게이머 픽셀"이 정확하게 정리해주는 게임 블로그. 다크 네온 그린 테마.

## 다루는 카테고리
- 게임 추천(RPG·협동·파티·플랫폼별), 공략·설정(FPS 설정·입문 가이드), 스팀·플랫폼 가이드(환불·세일), 콘솔 정보(스위치·스위치2·하위호환), 게임 뉴스·신작 등.

## 다른 게임 글과의 차별점
1. **상황·취향별로 갈래를 나눠 판단 기준을 준다** (입문/숙련, 혼자/여럿, 기기별, 시간·예산별).
2. **1인칭 '직접 파보는 게이머' 시점** — 남의 정보 복붙이 아니라 직접 찾아보고 정리한 경험.
3. **추정은 추정이라고 명시** — 사양·가격·패치·출시일은 단정 안 함, 시점·버전 변수 밝힘.

## EEAT 신뢰 근거
- 공식 출처 우선(스팀 공식·닌텐도 공식·개발사 공식 등) + 외부링크로 검증 가능.
- 잘 바뀌는 정보(가격·세일·패치·사양)는 집계 시점·추정 여부 표기.

## CTA 방침 (중요)
- **현재 제휴 미설정 → 구매·제휴 버튼 절대 금지.** (쿠팡파트너스 등 승인·설정 전이면 `.btn` 제휴 링크 금지.)
- 마무리 CTA는 "관련 글 더 보기"형 내부링크로만. 다음 글로 이어지게.
