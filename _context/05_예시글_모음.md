# 05. 예시 글 모음 (가장 강력한 레퍼런스)

픽셀 톤이 잘 적용된 글을 여기에 모은다. 새 글 쓸 때 "이 톤·문장 길이·구조를 almost mimic 하라"고 지시.

## 페르소나 적용 1호 (예정)
- 아직 '픽셀' 페르소나를 1인칭 변수까지 의식적으로 입힌 글은 없다.
- **페르소나로 첫 글을 쓰면 여기 경로를 등록한다.** (예: `src/content/blog/<slug>.md` — '픽셀' 페르소나 + 1인칭 변수 4~6회 처음 적용)
  - 본받을 점이 될 요소: 도입을 "찾아보면 정보가 다 달라 헤맸다"는 1인칭으로 열고, 중간에 "직접 표로 묶어봤다" 경험을 섞되 사실·수치는 추정 표기 유지.

## 후보 레퍼런스 (구조·톤 잘 잡힌 기존 글)
- **`src/content/blog/rpg-game-recommendations.md`** — 아직 1인칭 변수는 약하지만, 존댓말 톤·`cg-summary`/`cg-tip` 박스·취향별 H2 갈래·한눈 비교표·FAQ 5개의 구조 리듬이 깔끔하다. 새 글의 박스/표/FAQ 리듬은 이 글을 따른다.
- (참고) `src/content/blog/switch-game-recommendations-2026.md`, `steam-refund-guide.md` — 상황별 비교표·콜아웃·FAQ 활용이 좋은 추가 참고.

## 추가 예정
- 페르소나로 새로 쓰거나 리라이트한 글이 나오면 여기에 경로를 계속 추가한다.
- 가장 잘 나온 2~3편만 유지(너무 많으면 톤이 흐려짐).
