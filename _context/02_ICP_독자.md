# 02. ICP · 독자

## 핵심 독자 1줄
게임을 좋아하지만, 검색하면 공략·추천·사양·정책이 출처마다 달라 헷갈려서 "정확히 정리된 한 편"을 찾는 한국 게이머.

## 들어오는 검색 의도
- 추천 목록 — "RPG 게임 추천", "스위치 2인 게임", "협동 게임 추천".
- 방법(How-to) — "스팀 환불 방법", "FPS 설정 기본값", "게임 닉네임 짓는 법".
- 정보·규칙 이해 — "스위치 하위호환", "스팀 세일 시기", "게임 장르 차이".
- 비교 — "스위치 vs 스위치 2", "PC vs 콘솔", "장르별 입문작 비교".

## 독자 페인포인트 5개
1. 검색하면 글마다 추천·정보가 달라서 뭐가 맞는지 모르겠다.
2. 영어·전문 용어를 그대로 옮겨놔서 안 읽힌다.
3. 단편적 나열만 있고 "내 상황엔 뭐가 맞는지" 판단 기준이 없다.
4. 출처 없이 단정해서(특히 사양·가격·패치) 믿어도 되는지 불안하다.
5. 글이 길기만 하고 정작 궁금한 답이 위에 없다.

## 독자가 결정 직전 궁금해하는 것
- "내 기기·내 취향엔 뭐가 맞아?" (상황별 갈래)
- "그래서 결론이 뭔데?" (3줄 요약 욕구)
- "이거 진짜 최신 정보 맞아?" (출처·시점·추정 표기)

## 읽고 나서 바라는 행동
- 제휴 미설정 상태 → 지금은 **버튼·구매 유도 금지**.
- 목표는 "내 상황에 맞는 답을 정확히 알았다"는 만족 → 내부링크로 다른 글 회유 → 재방문.
